# IceCreamParlour-Management-System

## 📌 Project Overview

The **Ice Cream Parlour Management System** is a desktop application developed in **C# (Windows Forms)** using the .NET Framework. It is designed to automate and manage the daily operations of an ice cream shop, including customer handling, order processing, inventory management, employee records, and billing.

This system provides a user-friendly interface and helps improve efficiency by replacing manual record-keeping with a digital solution.

---

## 🎯 Objective

The main objective of this project is to:
- Automate ice cream shop operations
- Reduce manual workload and human errors
- Provide fast and accurate billing and record management
- Improve business efficiency through digital management

---

## 🧩 Features

### 👤 Customer Management
- Add new customers
- View and update customer records
- Maintain customer history

### 🍦 Order Management
- Create new orders
- Select items from menu
- Automatic price calculation

### 📦 Stock Management
- Add new ice cream items
- Update stock availability
- Track inventory status

### 👨‍💼 Employee Management
- Add and manage employees
- Store job roles and salaries
- Update or remove employee data

### 💳 Billing System
- Automatic bill generation
- Tax calculation
- Printable invoice support

### 🔐 Login System
- Secure admin authentication
- Access control for system protection
